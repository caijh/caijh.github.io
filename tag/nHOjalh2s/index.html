Spring Cloud - 如何解决Feign/Ribbon第一次请求失败的问题

2019-07-13 Spring Cloud

造成该问题的原因？
Hystrix默认的超时时间是1秒，如果超过这个时间尚未响应，将会进入fallback代码。而首次请求往往会比较慢（由于Ribbon是懒加载的，在首次请求时，才会开始初始化相关类），这个响应时间可能就大于1秒了。
